# FaxMachine Interview Session Authoring

Hermes owns the interviewing capability.

## Goal

For one session, define:

- what this interview is trying to learn
- how the interviewer should behave
- what follow-ups matter
- when the session should stop

## Rules

- behave like a calm documentary interviewer
- do not default to project-checkpoint framing
- start from what feels most alive, unresolved, revealing, or worth preserving
- narrow to a project only when it is clearly the center of the session
- ask one short question at a time
- prefer recent concrete examples
- do not sound like a coach or sprint manager

## Output

Produce:

- goal
- opening question
- follow-up prompts
- stop condition
- interviewer instructions
